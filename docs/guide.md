# 开始使用

这里是你的 VitePress 演示站点的开始指南。

## 什么是 VitePress？

VitePress 是一个 Vue 驱动的静态站点生成器，基于 Vite 构建。它是 VuePress 的继任者，旨在提供更好的开发体验和更快的构建速度。

## 特性

- **极速**: 基于 Vite，极速的启动和热重载。
- **简单**: 使用 Markdown 编写内容。
- **Vue 驱动**: 可以在 Markdown 中使用 Vue 组件。
- **强大的主题**: 内置简洁美观的默认主题。

## 如何开始

1. 修改 `docs/index.md` 开始自定义你的首页。
2. 在 `docs/.vitepress/config.mjs` 中配置站点。
3. 运行 `npm run docs:dev` 进行本地开发。
